Функция вычисления итоговой стоимости одежды

Returns
Стоимость всех вещей в Ателье

Данная функция вычисляет итоговую стоимость всех вещей в ателье по формуле: \(first.CostProduct + second.CostProduct() + more\).

Код функции выглядит следубщим образом:

public float TotalCost() { return first.CostProduct() + second.CostProduct() + more; }
